# FizTarefa
Fiz Tarefa é um aplicativo web simples e funcional desenvolvido com React, criado para ajudar usuários a organizar suas tarefas diárias, marcar atividades concluídas e melhorar sua gestão de tempo. Com uma interface intuitiva, o app permite adicionar, visualizar e remover compromissos, promovendo foco e produtividade no dia a dia.
